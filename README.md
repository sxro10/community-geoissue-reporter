# Community Problem Reporter (Java GUI)

A simple Java desktop application that lets users record community problems with description, location, severity, and status. The app provides a basic GUI to add reports, view all reports, filter by severity, update status, and save or load data from a local file.

## How to Run

1. Open the project in your Java IDE (Eclipse, IntelliJ, or similar).
2. Ensure JDK 8 or higher is installed.
3. Compile the project.
4. Run the `MainApp` class from the `com.community` package.
5. A window titled “Community Problem Reporter” will open. Use the form fields and buttons to interact with the app.

## Features

- Add new community problem reports with description, location, and severity.
- View all saved reports in a scrollable text area.
- Update the status of any report by its ID.
- Filter reports by severity (All, Low, Medium, High).
- Save reports to a local text file and load them again later.
